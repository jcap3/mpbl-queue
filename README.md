# Queue Apps

Single-file, mobile-first web apps for managing fair pickup-game rotations. Each project is self-contained — open its `index.html` in a browser.

## Projects

- [**basketball-queue**](./basketball-queue) — 5v5 basketball pickup rotation (10 players/game, 16-min timer, fair sit-out priority).
- [**pickle-queue**](./pickle-queue) — *(coming soon)* pickleball rotation.
